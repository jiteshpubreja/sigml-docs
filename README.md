# SiGML Docs
## This repo contains Unofficial Documentation for the SiGML Language.
#
#### [SiGML](https://vh.cmp.uea.ac.uk/index.php/SiGML) (Signing Gesture Markup Language) is a XML application that enables transcription of sign language gestures.
#### SiGML is further extended by [SiGML Extensions](https://vh.cmp.uea.ac.uk/index.php/SiGML_Extensions).
